# Data Set Description

The model operates on a rolling 90-day transaction graph constructed from **12M+ daily domestic and cross-border transfers**. Node features are derived from account metadata; edge features from transaction attributes.

| Component | Volume |
| --- | --- |
| Active Nodes (accounts) | ~850,000 |
| Daily Edges (transactions) | ~12,000,000 |
| Confirmed SAR clusters (labels) | 1,240 |
